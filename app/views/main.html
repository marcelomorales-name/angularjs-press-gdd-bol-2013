Desarrollo rápido con AngularJS

AngularJS combina elementos de HTML y Javascript para simplificar significativamente el desarrollo de aplicaciones ricas en la web y la comunicación con un back-end. Puede tomar el lugar de un marco de trabajo completo con documentación accesible y una brillante y vibrante comunidad. La charla está destinada a diseñadores, desarrolladores, estudiantes y cualquiera con alguna idea de HTML y Javascript.

Explicando HTML5

  • Navegadores compatibles
  • Puede comunicarse directamente con bases NoSQL
  • Diseñado para aplicaciones y no para documentos
  • Introduce web más semántica, por ejemplo <article>, o <menu>
  • Varios APIs, por ejemplo Web storage
  • Enlace a datos
  • Separación de Controller
  • Usa Javascript simple
  • Validación
  • Comunicación con el servidor
  • Componentes re-usables
  • Testeable
